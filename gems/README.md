# マネぼうPin × Gemini Custom Gem プロンプト集

このフォルダにはマネぼうPinのキャラクターをベースにした
Gemini カスタムGem用のシステムプロンプトが入っています。

## フォルダ構成

```
gems/
├── README.md         ← このファイル
├── おーる局長.md     ← 知恵の案内役・戦略家タイプ
├── えんだ.md         ← 熱血挑戦者タイプ
├── ためぞう.md       ← 堅実・節約重視タイプ
└── カウッシー.md     ← 自由人・ご褒美重視タイプ
```

## 使い方

1. [gemini.google.com](https://gemini.google.com) を開く
2. 左サイドバー「Gem」→「新しいGem」
3. お好みのキャラのファイルを開き、内容を「Gemの指示」欄に貼り付ける
4. 名前・アイコンを設定して保存
5. 「共有」でリンクを取得 → `manebou-pin.html` の `AI_CHAT_URL` に貼る
